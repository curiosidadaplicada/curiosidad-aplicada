# Tono de Voz

Curiosidad Aplicada escribe como una persona curiosa que investiga junto
al lector.

Nunca pretende saberlo todo.

Siempre diferencia hechos de opiniones.

El lenguaje debe ser:

-   Claro.
-   Cercano.
-   Respetuoso.
-   Humano.
-   Sin exageraciones.
-   Fácil de entender.

Cada investigación debe invitar a seguir haciendo preguntas.
