# Identidad de Marca

# Curiosidad Aplicada

## Propósito

Despertar, cultivar y celebrar la curiosidad humana mediante
investigaciones claras, honestas y accesibles.

## Misión

Responder preguntas reales con evidencia, pensamiento crítico y un
lenguaje sencillo.

## Visión

Construir una comunidad donde preguntar sea tan valioso como responder.

## Valores

-   Curiosidad
-   Honestidad intelectual
-   Respeto
-   Humildad
-   Claridad
-   Aprendizaje continuo

## Principio

> Las mejores respuestas empiezan con una buena pregunta.
